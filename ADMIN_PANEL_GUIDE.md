## Функции Admin Panel

### 1. 📊 Statistika (Статистика)
Показывает основные метрики:
- Количество пользователей
- Количество заказов
- Общий доход

### 2. 💰 Daromad (Калькулятор доходов)

**Это главная функция!** Здесь ты можешь:

#### Входные данные:
- **Sotilgan Stars soni** - Количество проданных звёзд (например, 100)
- **Narx (so'm/star)** - Цена за одну звезду в сумах (например, 195)
- **TON/USD kurs** - Курс TON к доллару (например, 3.20)
- **USD/UZS kurs** - Курс доллара к сумам (например, 12800)
- **Fragment komissiya (%)** - Комиссия Fragment (например, 5%)

#### Результаты расчёта:
- **Jami daromad (so'm)** - Общий доход в сумах
- **Stars soni** - Количество звёзд
- **TON narxi (USD)** - Цена TON в долларах
- **TON/UZS kurs** - Курс TON к сумам
- **Tannarx (so'm/star)** - Себестоимость одной звезды в сумах
- **Jami tannarx (so'm)** - Общая себестоимость в сумах
- **Sof daromad (so'm)** - **ЧИСТЫЙ ДОХОД** (Доход - Себестоимость - Комиссия)

### Пример расчёта:

**Входные данные:**
- Продано: 100 звёзд
- Цена: 195 сум/звезда
- TON/USD: 3.20
- USD/UZS: 12800
- Комиссия: 5%

**Результаты:**
- Общий доход: 19,500 сум
- Себестоимость: ~12,800 сум (примерно)
- Комиссия: 975 сум
- **Чистый доход: ~5,725 сум**

## Как использовать калькулятор доходов

1. Откройте Admin Panel
2. Введите пароль: `kama5505`
3. Перейдите на вкладку "💰 Daromad"
4. Введите количество проданных звёзд
5. Введите цену за звезду
6. Введите текущие курсы (TON/USD и USD/UZS)
7. Введите комиссию Fragment
8. **Смотрите результат в разделе "Sof daromad"**

## Автоматический расчёт

Калькулятор автоматически пересчитывает результаты при изменении любого значения.

## Формула расчёта

```
Общий доход = Количество звёзд × Цена за звезду

Себестоимость = Количество звёзд × (0.01 TON × (TON/USD × USD/UZS))

Комиссия = Общий доход × (Процент комиссии / 100)

Чистый доход = Общий доход - Себестоимость - Комиссия
```

## Примечания

- Все расчёты выполняются в реальном времени
- Курсы можно обновлять вручную
- Результаты показываются в сумах (UZS)
- Комиссия Fragment обычно составляет 5%
