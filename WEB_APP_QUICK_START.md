# 🌐 Web App - Быстрый старт

## Что это?

Красивое веб-приложение для покупки Telegram Stars и отправки подарков прямо из бота.

## Как открыть?

### Вариант 1: Из бота (лучше всего)
Нажмите на кнопку "🌐 Web App" в боте

### Вариант 2: Прямая ссылка
```
https://your-domain.com/app.html
```

## Что можно делать?

### 1️⃣ Купить звёзды
- Выберите количество (50, 100, 250, 500, 1000, 2500, 5000)
- Цена: **195 сум за звезду**
- Звёзды отправляются на ваш Telegram

**Примеры:**
- 50 ⭐ = 9,750 сум
- 100 ⭐ = 19,500 сум
- 1000 ⭐ = 195,000 сум

### 2️⃣ Отправить подарок
- Введите username получателя (например: @admin)
- Выберите подарок из 17 вариантов
- Цена: **50,000 сум за подарок**

**Доступные подарки:**
🐻 🎁 ❤️ 🌹 🎂 🌸 🚀 ✨ 💍 💎 🥤 👑 🎪 🎭 🎨 🎸 🎯

### 3️⃣ Смотреть историю
- Все ваши покупки и отправки
- Дата, время и сумма

## Как это работает?

```
1. Откроете web app
   ↓
2. Приложение получает ваш Telegram ID
   ↓
3. Загружает ваш баланс
   ↓
4. Вы выбираете что купить
   ↓
5. Деньги списываются с баланса
   ↓
6. Звёзды/подарки отправляются
   ↓
7. История обновляется
```

## Требования

- ✅ Telegram аккаунт
- ✅ Баланс в боте (пополните через /topup)
- ✅ Интернет соединение

## Безопасность

- 🔒 Все данные передаются через HTTPS
- 🔒 Telegram Web App проверяет подпись
- 🔒 Баланс хранится на сервере
- 🔒 Никакие пароли не требуются

## Если что-то не работает

### "Ошибка загрузки баланса"
→ Перезагрузите приложение (F5 или свайп вниз)

### "Недостаточно средств"
→ Пополните баланс в боте (/topup)

### "Ошибка отправки подарка"
→ Проверьте что username правильный (без @)

keyboard = InlineKeyboardMarkup(inline_keyboard=[
    [InlineKeyboardButton(
        text="🌐 StarPay Web App",

    )]
])

await message.answer("Откройте web app:", reply_markup=keyboard)
